en-IN on XP SP3?

by Michael S. Kaplan, published on 2011/09/08 07:01 -04:00, original URI: http://blogs.msdn.com/b/michkap/archive/2011/09/08/10207673.aspx

There is a new common quesion I am geting asked these days:

Hi Team,

Is it possible to  have English (India) for xp sp3 under Regional and Languages options? Is there a way to achieve this?

However we can this option with windows 7.

Any help is highly appreciated.

Thanks & Regards,

If there were only a way.

There were two features added to Vista, and present in all versions after it:

  • The actual en-IN locale was added, and
  • The ability to create custom locales was added.

But for all earlier versions of Windows, or managed code a custom culture will work (though it cannot be installed without extra work by Locale Builder). But it cannot be used by Windows itself.

Not even on XP SP3....
