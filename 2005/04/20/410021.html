Upcoming speaking gigs in Cleveland, OH

by Michael S. Kaplan, published on 2005/04/20 10:00 -04:00, original URI: http://blogs.msdn.com/b/michkap/archive/2005/04/20/410021.aspx

The first presentation is at the Microsoft SQL Server Special Interest Group. It is happening tomorrow night (April 21st) at 6:30pm.

The second presentation is at the Microsoft C#/VB.NET SIG on Monday night (April 25th) at 6:30pm.

The first talk is about the internationeal features in SQL Server, and the second is about the globalization features in the .NET Framework in its newest release (Whidbey, whose Beta 2 is now available).

Are you in the Cleveland area? Pop by for either or both talks, and come up to introduce yourself afterwards!
