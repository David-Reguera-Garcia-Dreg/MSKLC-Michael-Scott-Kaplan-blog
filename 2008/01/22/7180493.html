Best possible support before there was support?

by Michael S. Kaplan, published on 2008/01/22 10:01 -05:00, original URI: http://blogs.msdn.com/b/michkap/archive/2008/01/22/7180493.aspx

Raja asks via the Contact link:

HI,

I am using Windows 98 Second Edition as my OS. Pl suggest how to view MSN site in Tamil.

Thank you.
raja

On the whole, the only real answer here is an upgrade since a lot of the basic language support needed for Tamil is only available in Windows 2000 and later.

But some prior versions of Internet Explorer provided (vis MLang and Uniscribe) a limited amount of support for "Unicode-only" languages such as Tamil that would work on Windows 98 and Windows Me. Getting the most recent version of Internet Explorer that will install on the machine is a way to get things as best as they can be....

Still missing is a bunch of the support one really wants on the input/typography/globalization side, even with the best effort work that went into MLang and Uniscribe.

Windows 98 just came before Windows was really looking at this scenario....
